## Features

- Astro 4.x + React 18
- TypeScript strict mode
- Modular SCSS styling
- Docker Compose for dev/prod environments

## Installation

```bash
git clone https://github.com/pardnchiu/astro-startup.git
cd astro-startup
cp .env.example .env
```

## Usage

### Docker (Recommended)

```bash
# Development
docker compose up

# Production
docker compose -f docker-compose.prod.yml up
```

### Local Development

```bash
cd app
npm install
npm run dev
```
